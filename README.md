## 📖 Sobre o Projeto

O **DaviDev** é uma aplicação web desenvolvida com **Django**, criada para servir como meu portfólio profissional e centralizador de projetos. O objetivo é apresentar minhas aplicações, habilidades e tecnologias utilizadas ao longo da minha jornada como desenvolvedor.

O projeto foi construído seguindo boas práticas de desenvolvimento web, oferecendo uma interface moderna, responsiva e otimizada para produção. A aplicação está hospedada no **Render** e utiliza o **PostgreSQL** como banco de dados principal, garantindo maior desempenho, escalabilidade e confiabilidade.

🔗 **Acesse o projeto:**  
**https://davidev.onrender.com**

---

## ✨ Funcionalidades

- 💼 **Portfólio Dinâmico**
  - Exibição organizada dos projetos desenvolvidos.

- 📱 **Interface Responsiva**
  - Compatível com computadores, tablets e smartphones.

- 🖼️ **Gerenciamento de Imagens**
  - Upload e renderização otimizada das imagens dos projetos.

- 🗂️ **Arquitetura Modular**
  - Organização baseada no padrão MVT do Django.

- ⚡ **Deploy em Produção**
  - Aplicação publicada na plataforma Render.

- 🛠️ **Banco de Dados PostgreSQL**
  - Armazenamento seguro e eficiente das informações da aplicação.

## 🏗️ Estrutura do Projeto

```text
📦 DaviDev
│
├── core/                # Configurações do projeto Django
├── fusion/              # Aplicação principal
├── templates/           # Templates HTML
├── static/              # Arquivos CSS, JavaScript e imagens
├── media/               # Uploads de imagens
│
├── build.sh             # Script de deploy
├── Procfile             # Inicialização do Gunicorn
├── requirements.txt     # Dependências do projeto
├── runtime.txt          # Versão do Python
└── manage.py
```

## 🗄️ Banco de Dados

O projeto utiliza o **PostgreSQL** como banco de dados principal, proporcionando:

## ☁️ Deploy

A aplicação está hospedada no **Render**, utilizando:

- Gunicorn como servidor WSGI;
- PostgreSQL como banco de dados;
- Gerenciamento de arquivos estáticos para ambiente de produção.
